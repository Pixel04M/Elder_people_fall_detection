Elder People Fall Detection System

An IoT-based fall detection system designed for elderly people. Using ESP32 and Arduino IDE, the system processes sensor data to detect falls and immediately sends real-time alerts via an Android mobile application. This project aims to enhance safety and provide quick assistance in emergency situations.

Tech Stack: ESP32, Arduino IDE, IoT, Sensors, Android App Development
